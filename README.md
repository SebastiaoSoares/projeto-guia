# GUIA - Gestão Unificada de Integração e Acolhimento

O projeto **GUIA** é uma plataforma focada na transformação do processo de *onboarding* de novos colaboradores nas empresas, oferecendo informação e acolhimento. O objetivo é aumentar a produtividade da gestão, do RH, dos demais setores envolvidos e, claro, do colaborador recém-chegado.

A motivação surgiu da necessidade que muitas empresas têm de organizar o processo de integração através de um **checklist único e compartilhado** entre os diferentes setores. O RH é uma parte essencial, bem como o SESMT, a TIC, o Setor Pessoal e várias outras partes interessadas em simplificar seus processos por meio de uma única fonte de verdade e transparência durante a admissão.

## Funcionalidades Principais
* Checklist compartilhado em tempo real
* Dashboard para os setores envolvidos no processo de onboarding
* Clareza e transparência com informações comaprtilhadas entre setores e gestor do novo colaborador
* Manual do novo colaborador com informações relevantes

## Tecnologias Utilizadas
* HTML/CSS/JS
* Python/Flask
* SQLite

## Contribuindo com o projeto

### Pré-requisitos
Certifique-se de ter o [Python 3.x](https://www.python.org/) instalado em sua máquina.

### Inicializando o ambiente

1. Clone o repositório e acesse a pasta do projeto.
2. (Recomendado) Crie e ative um ambiente virtual:
```bash
 python -m venv venv

 # No Windows:
 venv\Scripts\activate
 # No Linux/Mac:
 source venv/bin/activate
```
3. Instale as dependências:
```bash
pip install -r requirements.txt
```
4. Execute a aplicação Flask:
```
python app.py
```

A aplicação estará disponível em `http://localhost:5000`.
